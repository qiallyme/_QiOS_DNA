# QiSystem Overview

QiSystem is the operational integrity layer. It tracks the evidence that the system is running correctly and keeps generated operational material separate from product, portal, and doctrine docs.

QiSystem supports the active model:

```text
QiAccess -> QiLife -> (QiSystem + QiNexus + QiCapture + QiConnect)
```

It is the place for logs, audits, backups, health checks, generated reports, generated indexes, and maintenance notes.
