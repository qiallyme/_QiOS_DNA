# QiLife

QiLife is the living operational data spine and AI LifeDesk of QiOS.

## Working loop

```text
Capture -> Agent Draft -> Review -> Save -> Timeline -> Dashboard / Actions / Recent QiBits
```

## Boundary

QiLife owns captured life records, QiBits, actions, events, people, transactions, obligations, documents, knowledge items, links, activity logs, AI outputs, and summaries.

QiLife does not own the full QiOS doctrine. That belongs to QiOS DNA.

QiJourney may interpret QiLife records, but interpretation is derived unless tied back to canonical records.
