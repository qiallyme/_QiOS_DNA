# QiApp QiLife Flows

## QiBit Lifecycle

```text
Capture
  -> Bucket
  -> Interpret
  -> Relate
  -> Slot
  -> Breakdown
  -> Enrich
  -> Act
  -> Resolve
  -> Reflect
  -> Retrieve
```

## Timeline Flow

Timeline is a projection over timestamped records, not a separate table in v1.

```text
QiBit, Action, Transaction, Event, Daily Summary
  -> timestamp rule
  -> timeline view
```

## Knowledge Flow

```text
Markdown source
  -> indexed/imported into QiLife knowledge
  -> shown through Context Dock
  -> linked back to source
```
