# Actions: UI Flow
1. **Lists & Kanbans**: Cody views Actions filtered by states: Today, Next, Waiting, Scheduled, and Done.
2. **Action Detail Pane**: Displays Action details, description, due date, priority, energy level, and associated **Steps**.
3. **Toggle Step**: Cody checks off individual steps, which updates the step's completion timestamp in the UI.
4. **Triage to Next**: Actions can be quick-shifted to "Today" or rescheduled via context menu clicks.
