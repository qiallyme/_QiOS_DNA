# QiNexus Flows

## Placement Flow

```text
Item needs storage
  -> identify bucket/domain
  -> place in matching QiNexus location
  -> link from QiLife record when applicable
  -> archive only when inactive or historical
```

## App Alignment Flow

```text
QiLife bucket
  -> matching QiNexus folder
  -> files, references, and exports stay in the same domain
```

Tags support retrieval. Links and folder placement define known structure.
