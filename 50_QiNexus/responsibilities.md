# QiNexus Responsibilities

QiNexus is responsible for:

- Folder discipline.
- Durable storage categories.
- Bucket alignment for QiLife.
- Archive placement.
- Reference material placement.
- Data exports and structured datasets.
- Keeping files findable without conceptual guessing.

QiNexus is not responsible for:

- App workflow logic.
- Runtime hosting.
- Portal navigation.
- Doctrine.
