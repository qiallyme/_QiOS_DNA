## License Templates

### MIT License
- **Template:** `LICENSE_TEMPLATE_MIT.md`
- **Best for:** Most websites, apps, and personal projects
- **Simple:** Easy to use, minimal restrictions
- **Usage:** Copy to project root as `LICENSE`, replace `[YEAR]` and `[COPYRIGHT HOLDER]`

### Apache License 2.0
- **Template:** `LICENSE_TEMPLATE_APACHE.md`
- **Best for:** Business projects, enterprise software, projects requiring patent protection
- **Features:** Patent grant, contributor-friendly
- **Usage:** Copy to project root as `LICENSE`, replace `[YEAR]` and `[COPYRIGHT HOLDER]`
