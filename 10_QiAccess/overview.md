# QiAccess Overview

QiAccess is the portal and navigation shell for the system. It is the daily entry point for opening tools, seeing what needs attention, capturing quickly, and reaching system services.

In the active model:

```text
QiAccess -> QiLife -> (QiSystem + QiNexus + QiCapture + QiConnect)
```

QiAccess is the cognitive front door. It should reduce search, reduce friction, and make the right next surface obvious.

The active portal roots are:

1. Home
2. Start
3. Capture
4. Knowledge
5. Memory
6. Insights
7. System
