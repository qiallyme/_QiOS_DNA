# QiAccess Structure

QiAccess active navigation has seven roots:

```text
Home
Start
Capture
Knowledge
Memory
Insights
System
```

System subroutes stay nested under System:

- access
- server
- storage
- integrations
- settings
- blueprint
- roadmap
- security
- diagnostics

The live portal is currently represented by the Homepage-powered QiAccess Start runtime on QiServer.
