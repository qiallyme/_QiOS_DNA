# Agent-First Product Doctrine
QiLife is an AI Life Copilot with a Personal LifeDesk cockpit.
- Agent-first, not app-first. QiLife should behave like Cody’s AI copilot inside an app.
- The user should be able to talk/type/brain-dump to QiLife and have it extract QiBits, actions, obligations, reports, insights, and relevant context.
- Manual CRUD screens are secondary control surfaces, not the primary workflow.
- Main interaction is: “What happened?” / “Talk to QiLife” / “Handle this.”
