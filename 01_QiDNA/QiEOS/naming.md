# QiEOS Naming

System layer names are fixed:

- QiDNA
- QiAccess
- QiSystem
- QiServer
- QiCapture
- QiNexus
- QiApp QiLife
- QiConnect

Use these names exactly in active documentation. Do not reintroduce older folder names such as `qicore`, `qiarchive`, `architecture`, `modules`, or `product` as top-level active structures.

QiLife object names:

- QiBit: one captured unit of life data.
- Bucket: top-level categorization domain.
- Thread: an ongoing case, project, storyline, or situation.
- Action: a task or work order.
- Step: a subtask inside an action.
- Context Dock: the side context layer attached to work.
