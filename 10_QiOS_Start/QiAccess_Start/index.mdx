# QiAccess Start

QiAccess Start is the QiOS front door, cockpit, launcher, access dashboard, and docs surface.

## Role

QiAccess Start answers:

```text
Where do I go next, and how do I safely get there?
```

It surfaces tools, docs, service links, server entry points, current priorities, and operating dashboards.

## Boundary

QiAccess Start is not the whole operating system.

It is the interface layer for QiOS.

## Related layers

- Runtime belongs to QiServer.
- Doctrine belongs to QiEOS / QiOS DNA.
- Rules and schemas belong to QiSystem.
- Storage belongs to QiNexus.
